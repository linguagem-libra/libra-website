Linguagem Libra

Libra é uma linguagem de programação simples, criada exclusivamente para ser um projeto acadêmico. Este projeto tem como objetivo principal o aprendizado e a experimentação no desenvolvimento de linguagens de programação. O código fonte da linguagem estará disponível publicamente para consulta, uso e contribuição.

Instalação

Você pode instalar a ultima versão disponível da Libra abaixo:

Libra-Win64 (Windows 64bit)

Código Fonte (Código Fonte da Libra)

Baixe e extraia os arquivos em uma pasta no diretório desejado, depois adicione o diretório no PATH do Windows, veja um exemplo aqui.

Existe um manual em PDF que contém tudo o que você precisa saber para programar usando a Libra.
